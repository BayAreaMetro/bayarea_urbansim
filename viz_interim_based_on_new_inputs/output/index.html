    Diagnostic Outputs

    The table below contains brief descriptions of the disgnostic output tables of BAUS.

    File name Purpose File type
    simulation_output.json Two sets of zone-level attributes: one set is the same as the standard Travel Model output, the other set is diagnostic attributes representing development capacity, pricing, logsum, vacancy, development type and quantity, etc., used to help analyze/debug submodels. JSON
    parcel_output.csv Parcel-level attributes of parcels with development activities during the simulation time period. CSV
    dropping_buildings.csv XXX CSV
